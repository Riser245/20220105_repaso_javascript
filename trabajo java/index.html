Metodos de array en javascript

Arreglo reduce

El método reduce() en JavaScript es una función de orden que se utiliza para reducir los elementos de un arreglo a un soo valor. Permite revisar sobre cada elemento del arreglo y combinarlos en un valor acumulado.

Arreglo filter

El método filter() en JavaScript es una función que se utiliza para crear un arreglo nuevo a partir de otro arreglo, manteniendo solo los elementos que cumplan cierta condición. Permite filtrar los elementos de un arreglo según un criterio especificado y devuelve un nuevo arreglo con los elementos que pasaron dicha condición.
